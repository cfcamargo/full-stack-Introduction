## Para Rodar a aplicação Front End
<br>

- Primeiro instalamos as dependencias com: 
```
  npm install

```

- Depois de instalar as dependencias podemos executar a Aplicação com:
```
  npm run dev

```

- Para compilar para produção usamos :
```
  npm run build

```
